# 🍽️ MealForAll AI

An AI-powered intelligent food rescue and redistribution platform that connects surplus food donors with nearby NGOs using real-time location matching, preparation & expiry tracking, and machine learning-based food quality detection.

---

## 🚀 Features

- 📍 Location-based food donation matching  
- ⏳ Food preparation & expiry tracking  
- 🤖 AI-based food quality detection (Fresh / Near Expiry / Expired)  
- 🔔 Real-time notifications for NGOs  
- 🚚 Smart food redistribution system  

---

## 🛠️ Tech Stack

- Frontend: Next.js / React  
- Backend: Python (FastAPI)  
- Database: (your DB)  
- AI Model: TensorFlow / OpenCV
